F#

F# is a great language for functional programming. It combines functional programming concepts with terse readable syntax. Here's a very simple hello world function in F# 


 printfn "Hello World!"
Output: Hello World!

Of course, this is much too simple, we need a much more interesting example. Something F# excels in is recursion, here's an example of a recursive function: 


//Get the count of the list.
let rec getCount (list, count) =
    match list with
    | [] -> count
    | _ -> getCount (list.Tail, count + 1)
Consumes a list and returns the count of that list.

This function gets the count of a list you pass in. One thing to take notice of is the "list.Tail" function which gives you the list minus the first element.
